# 📝 Atividade em Sala: Lista de Tarefas com Streamlit

Este repositório contém uma atividade prática desenvolvida em sala de aula para exercitar os conceitos de desenvolvimento web com Python e gerenciamento de estado.

O projeto consiste em um aplicativo simples de **Lista de Tarefas** que permite adicionar, visualizar e excluir itens utilizando a biblioteca **Streamlit**.

---

## ✨ Funcionalidades

- ➕ Adicionar novas tarefas à lista.
- ✅ Visualizar tarefas com caixas de seleção (checkbox).
- ❌ Excluir tarefas individuais da lista.
- 🔄 Gerenciamento de estado temporário com `st.session_state` (evita que a lista suma ao atualizar a página).

---

## 🛠️ Tecnologias Utilizadas

- **Python**
- **Streamlit**
